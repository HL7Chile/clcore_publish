2. Practitioner.identifier
Definition

Identificador para cada prestador. El identificador principal en Chile es el Registro Único Nacional (RUN), identificador que es obligatorio. Además, se puede ingresar un ID extra para cada prestador, disponibilizado por Registro Nacional de Prestadores Institucionales (NRPI), este identificador es opcional

ShortId de los prestadores
Comments

Este elemento permite ingresar dos identificadores, uno de tipo RUN y uno de tipo RNPI

NoteThis is a business identifier, not a resource identifier (see discussion)
Control1..?
Must Supporttrue

  • value @ use
